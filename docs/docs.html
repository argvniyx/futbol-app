2 API

2.1 Funciones

Las funciones del API están descritas según las vistas de la página

2.1.1 Registro

  1. createParent(Padre) => Result Code
    • Does not exist and is successful
    • Does not exist but failed
    • Already exists
  2. createCoach(Entrenador) => Result Code
    • Does not exist and is successful
    • Does not exist but failed
    • Already exists
  3. createTeam(Team) => Result Code
    • Does not exist and is successful
    • Does not exist but failed
    • Already exists

2.1.2 Dashboard General

Estas son funciones que son comunes al dashboard de usuario y entrenador

  1. authPerson(email, password) => Result Code
    • Success: personId
    • Failure: other code (negative number, for example)
  2. getParent(parentId) => Parent || Null
  3. getCoach(coachId) => Coach || Null
  4. getTeam(teamId) => Team || Null
  5. getNextEvents(currentEventId) => [ Event ]
    • Guard against first event
  6. getPreviousEvents(currentEventId) => [ Event ]
    • Guard against last event
  7. getTelephoneDirectory(teamId) -> [ Person ]
    • Failure: empty list
    • Success: [ Person ]

2.2 Endpoints

2.2.1 Usuario

POST /Register:
Registrar nuevo usuario
POST /RegisterGoogle:
Registrar nuevo usuario con cuenta de Google
DELETE /DeleteAccount:
Borrar mi Cuenta
POST /LogIn:
Hacer login
POST /Login:
Hacer login con Google
PUT /Info:
Editar información personal

2.2.2 Administrador

POST /:
registra nuevos Admin.
DELETE /Coach:
borrar algun Admin

2.2.3 Entrenador

POST /Invitation:
genera una nueva invitación
PUT /InvitationStatus:
Actualizar el estatus de invitación (Aceptada/Rechazada)
PUT /Season:
Modifica el inicio y fin de una temporada
PUT /TeamColors:
Modifica los colores del equipo para desplegarlos en el ui
PUT /Contact:
Modifica información de contacto
POST /Coach:
registra nuevo entrenador (Administrador).
DELETE / Coach:
borrar algun entrenador (Administrador)

2.2.4 Events

GET /Week:
Obtiene los eventos de cierta semana (Para padres y entrenadores)
POST /Event:
Genera un nuevo evento. (Partidos, Entrenamientos, etc…) (Solo Entrenador)
PUT /Event:
Modifica los detalles de un evento. (Entrenador)
DELETE /Event:
Borra un evento (Admin)

2.2.5 Padres

POST /Hijo:
Registra un nuevo hijo
PUT /Contact:
Modifica información de contacto
PUT /Invitation Status:
Actualizar el estatus de Invitación (Aceptada/ Rechazada)
GET /HijoDashboard:
Cambia de vista de hijo

2.2.6 Equipo

crear un equipo (Admin)
GET /:
Visualiza el directorio del equipo
DELETE /:
